# Brain Tumor Detection AI

AI project for classifying brain MRI images into four classes:

- Glioma tumor
- Meningioma tumor
- Pituitary tumor
- No tumor

The project includes exploratory data analysis, preprocessing, model training, model evaluation, and Grad-CAM visualization to help explain which image regions the model focuses on when making predictions.

## Project Overview

This repository contains a notebook-based machine learning pipeline for brain tumor MRI classification. The workflow is split into four main notebooks:

| Notebook | Purpose |
| --- | --- |
| `01_EDA.ipynb` | Explore dataset distribution, image sizes, sample images, and pixel statistics |
| `02_Preprocessing_v2.ipynb` | Preprocess MRI images, crop ROI, normalize images, and prepare train/validation/test data |
| `03_Model_v3.ipynb` | Train and tune CNN and EfficientNetB0 models, including Colab training results |
| `04_Evaluation_v2.ipynb` | Evaluate trained models and generate metrics, confusion matrix, ROC curve, wrong predictions, and Grad-CAM |

## Model Results

Final tuned model results from Colab training:

| Model | Accuracy | Macro Precision | Macro Recall | Macro F1 |
| --- | ---: | ---: | ---: | ---: |
| CNN Tuned | 0.7497 | 0.7551 | 0.7516 | 0.7438 |
| EfficientNetB0 Tuned | 0.8862 | 0.8866 | 0.8874 | 0.8854 |

The best-performing model in this experiment is `EfficientNetB0 Tuned`.

Training result files are stored in:

```text

```

This folder contains training curves, confusion matrices, comparison plots, and JSON metric summaries.

## Installation

Create and activate a Python environment, then install dependencies:

```bash
pip install -r requirement.txt
```

If using Conda:

```bash
conda create -n CSAI python=3.10
conda activate CSAI
pip install -r requirement.txt
```

## How To Run

Run the notebooks in this order:

1. `01_EDA.ipynb`
2. `02_Preprocessing_v2.ipynb`
3. `03_Model_v3.ipynb`
4. `04_Evaluation_v2.ipynb`

Recommended workflow:

- Use notebook 01 to inspect the dataset.
- Use notebook 02 to generate processed ROI images and split metadata.
- Use notebook 03 to train models. For long training, run this notebook on Google Colab.
- Use notebook 04 locally or on Colab after downloading `best_cnn_model.h5`, `best_tl_model.h5`, and the required data folders.

## Notes

- The model input uses 3-channel RGB tensors, although MRI images visually look grayscale.
- Grad-CAM heatmaps are visualization overlays, not original image colors.
- Model predictions with high confidence can still be wrong, so confusion matrix and per-class metrics should be checked alongside accuracy.
- This project is for educational and research purposes only. It is not a clinical diagnostic tool.
